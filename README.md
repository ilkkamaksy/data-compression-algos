## Run program

Run the app with

`./gradlew run`

## Performance tests

Run the performance tests with

`./gradlew run --args="compare"`

## Testing

Run tests with

`./gradlew test`

Create jacoco test coverage report with

`./gradlew jacocotestreport`

## JavaDoc

Create JavaDoc with

`./gradlew javadoc`


## Checkstyle

Run checkstyle with

`./gradlew check`
